Quick Start Guide

::

    from pygorithm.sorting import bubble_sort

    # This will print the code for bubble sort
    print(bubble_sort.get_code())

    myList = [12, 4, 2, 14, 3, 7, 5]

    # to sort the list
    sorted_list = bubble_sort.sort(myList)
